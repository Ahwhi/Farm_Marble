## 🎲 농장 마블(Farm Marble)

🏏 V1.0 (2025년 09월 12일)

● 본 게임은 **3D 온라인 보드 게임**으로, 첫 서버 적용 게임입니다.

● 멀티 게임에서 클라이언트와 서버간의 통신을 연습하기 위해 서버 공부 후 제작하였습니다.

---


### 🎮 게임 설명

☀️ 장르: **3D 온라인 보드 게임**

❄️ 플랫폼: Windows

⭐ 엔진: **Unity 2022. 3. 60f1**

💇‍♂ 프로젝트: **개인 (100%)**

🕜 제작 기간 : 2025. 08. 27 ~ 2025. 09. 12 **(16일간)**

---

### 🧾 기술 스택

🌟 **C# TCP 서버**: 회원가입, 로그인, 자동로그인, 채팅, 방 생성, 방 입장, 멀티 게임 등

🌟 **SQLite**: ID, Crypto 암호화 해시 패스워드 등 저장

🌌 **Unity**: 서버 권위형에 맞게 서버가 제공하는 값을 표기하는 방식으로 게임 구성
